# 🚗 Smart Parking Management System

A full-stack Smart Parking Management System built using the MERN Stack that allows users to register, log in, book parking slots, and manage bookings with an admin dashboard.

## 🌐 Live Demo

**Frontend:** https://smart-parking-anas.vercel.app/**
**Backend API:** https://smart-parking-system-660j.onrender.com**

## 🚀 Features

### User

- Secure Login

- Dashboard Analytics

## Installation

### Clone Repository

```bash
git clone https://github.com/hashmi-anas/smart-parking-system.git
```

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
